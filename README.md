# CleanJokes.com

CleanJokes.com is a simple coming soon landing page for a humor website focused on wholesome, family-friendly jokes.

## Overview

The site presents CleanJokes.com as a future destination for:

- Daily clean jokes
- Classic one-liners
- Dad jokes
- Silly puns
- Wholesome humor for all ages

The page emphasizes that the humor is:

- Safe for all ages
- Free of profanity
- Free of politics
- Privacy respectful

## Current Status

This is currently a **coming soon** page.

## Files

- `index.html` — main landing page
- `README.md` — project overview
- `sitemap.xml` — sitemap for search engines
- `robots.txt` — crawler instructions

## Design Notes

The landing page includes:

- A soft gradient background
- A large branded title
- A playful tagline
- A “Coming Soon” badge
- Several clean sample jokes
- A simple privacy-friendly footer

## SEO / Search Engine Notes

The site currently contains a single public page:

- `https://cleanjokes.com/`

The included `sitemap.xml` and `robots.txt` help search engines discover and crawl the homepage.

CleanJokes.com states that it does not track, sell, or misuse personal information.
